## 📸 Features

- 🎨 **Premium Design** — Elegant gold & warm color palette with modern typography (Cormorant Garamond + Montserrat)
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- 🖼️ **Interactive Gallery** — Filterable portfolio with lightbox viewer (Mandap, Stage, Floral, Royal, Outdoor)
- 📝 **Booking Form** — Availability checker with event type, budget range, and venue fields
- 💬 **WhatsApp Integration** — One-tap enquiry via WhatsApp
- ⭐ **Testimonials** — Client review showcase section
- 🗺️ **Google Maps** — Embedded location map in the footer
- 📞 **Sticky Mobile CTA** — Always-visible Call & WhatsApp buttons on mobile
- 🔍 **SEO Optimized** — Proper meta tags, semantic HTML, and descriptive alt text

## 📁 Project Structure

```
wedding/
├── index.html        # Main landing page (Home, Services, Gallery, Testimonials, Booking)
├── about.html        # About Us page
├── contact.html      # Contact page
├── styles.css        # Complete stylesheet with responsive design
├── script.js         # Interactive features (gallery filter, lightbox, form validation)
└── images/           # All website images (hero, gallery, logo, etc.)
```

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abhipatil2101/wedding-site.git
   ```

2. **Open in browser:**
   ```bash
   open index.html
   ```
   Or use a live server extension in VS Code for the best experience.
